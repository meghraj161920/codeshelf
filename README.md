# CodeShelf

CodeShelf is a digital marketplace platform where developers can upload, share, and sell ready-made software projects. The platform allows users to explore a wide range of coding projects such as web applications, AI tools, APIs, and mobile applications.

Users can browse projects by category, view project details, read reviews, and purchase downloadable source code. After purchasing, users can securely download the project files from their account.

CodeShelf is designed to help developers monetize their work while also helping learners and businesses quickly access high-quality, ready-to-use software solutions.

## Features

* User authentication system
* Project listing and category management
* Project screenshots and detailed project pages
* Secure project download after purchase
* Order and payment tracking
* Review and rating system
* Wishlist functionality
* Coupon and discount system
* User dashboard for managing purchases and downloads

## Tech Stack

* Backend: Django
* Frontend: HTML, CSS, Bootstrap
* Database: SQLite / PostgreSQL
* Version Control: Git & GitHub

## Database Structure

The platform includes the following main models:

* User
* Category
* Project
* ProjectImage
* Order
* OrderItem
* Review
* Coupon
* UserCoupon
* Wishlist
* ProjectDownload

## Purpose of the Project

The goal of CodeShelf is to create a platform similar to digital marketplaces where developers can distribute and sell their software projects. It also serves as a learning project demonstrating full-stack development using Django.
